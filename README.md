# OpenKM

OpenKM en contenedores Docker.

## Puesta en marcha

La primera vez, editar el fichero OpenKM.cfg y cambiar:

```
hibernate.hbm2ddl=create
```

Lanzar el servidor:

```shell
make start
```

Usuario por defecto: `okmAdmin/admin`.
